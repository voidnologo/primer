# Examples

Sanitized sample lesson artifacts, so a new user can see the quality and shape of what primer produces before running their first lesson.

These are frozen samples — not a learner's live record. Real lessons live in each user's **private** instance data repo (`$DATA_DIR/lessons/`), never in the public core. See `docs/engineering/GOALS.md` (privacy) and `primer/lesson-template.md` (the artifact contract).

- `event-driven-architecture/` — the outbox pattern (a worked lesson with retrieval prompts).
- `ai-agentic/` — agent patterns vs frameworks (a reference-style lesson).
